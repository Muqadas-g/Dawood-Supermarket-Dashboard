# Dawood Supermarket Dashboard 📈

A Python‑based interactive dashboard to analyze and visualize **Eid essentials sales** for *Dawood Supermarket, Nawabshah*.  
This project uses simple data and visual charts to help understand trends in sales, promotions, inventory, and external factors — perfect for decision making during Eid.

---

## 📌 Features

✔ Visualize yearly **sales trends** for key products (Dates, Rice, Oil, Beverages, Clothing, Gram Flour, Chickpeas)  
✔ Compare **sales with and without promotions**  
✔ View **current inventory levels**  
✔ Observe relationship between **price and sales**  
✔ Analyze impact of **external factors** (inflation, weather, competitor promotions)  
✔ See **top selling products** with pie chart visualization
